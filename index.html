Welcome to Dabom.org

Dabom.org is the official home of The Dabom Project — a collection of AI-powered accessibility tools designed for blind and visually impaired users.

The first two applications are Dabom v3.0 and Matheon. Dabom focuses on everyday life assistance through AI vision, screen reading, and audio output. Matheon focuses on study, research, and knowledge work in an accessible environment.

This website is built with accessibility first: clear headings, keyboard navigation, meaningful link text, and screen-reader-friendly structure.

The Dabom Project

The Dabom Project was created by Cheongrak with the belief that artificial intelligence can become a meaningful, respectful companion for people who experience the world primarily through non-visual means.

The project is committed to building tools that are practical, understandable, and humane — not impressive in isolation, but genuinely useful in everyday life.

The Dabom Project

The Dabom Project는 청락이 창립했습니다. 시각이 아닌 다른 방식으로 세상을 경험하는 분들에게 인공지능이 의미 있고 존중받는 동반자가 될 수 있다는 믿음에서 출발했습니다.

이 프로젝트는 단독으로 인상적인 기술이 아니라, 일상에서 진정으로 유용한 — 실용적이고, 이해하기 쉽고, 인간적인 도구를 만드는 데 헌신합니다.

About Dabom v3.0

Dabom is an AI-powered life assistant for blind and visually impaired users running on Windows. It helps users understand visual information that would otherwise be difficult or impossible to access independently.

Core Features

  • AI screen description — describe what is currently on screen
  • Camera-based description — describe objects, documents, faces, or surroundings
  • OCR and screen text recognition
  • Translation of screen content and clipboard text
  • AI question answering — ask anything about the screen or clipboard
  • YouTube caption reading and translation
  • File transcription — audio and video files
  • Live microphone and system audio transcription
  • PDF reading (summary and full text)
  • Video scene description
  • Color and brightness detection
  • Microsoft Neural Voices (high-quality TTS) support
  • SenseReader (Korean screen reader) integration

Language Support

Dabom v3.0 supports 9 interface languages: Korean, English, Spanish, French, German, Japanese, Chinese, Russian, and Arabic. The application automatically detects the OS language on first launch.

Designed for Real Independence

Every feature in Dabom is designed to be operated confidently by keyboard alone, with clear and immediate spoken feedback. The goal is not to demonstrate AI capability, but to reduce real barriers in daily life.

진정한 독립을 위한 설계

다봄의 모든 기능은 키보드만으로 자신 있게 조작할 수 있도록 설계되었으며, 즉각적이고 명확한 음성 피드백을 제공합니다. AI 기술을 과시하는 것이 목적이 아니라, 일상의 실질적인 장벽을 낮추는 것이 목표입니다.

About Matheon

Matheon is an AI-assisted learning and research tool for blind and visually impaired students, researchers, writers, and lifelong learners.

While Dabom focuses on daily life and visual accessibility, Matheon focuses on intellectual work — reading, writing, research, and organized knowledge management in a screen-reader-friendly environment.

Matheon Is Designed To Help With

  • Study and research tasks
  • Writing support and document preparation
  • Accessible interaction with educational materials
  • Screen-reader-friendly workflows
  • Keyboard-centered learning and productivity

마테온 (Matheon)은 시각장애인 학생, 연구자, 작가, 평생 학습자를 위한 AI 기반 학습·연구 도구입니다.

Why AI Accessibility Matters

Many everyday tasks still depend heavily on visual information — printed documents, signs, product labels, user interfaces, and visual surroundings can all create barriers for blind and visually impaired users.

AI cannot replace human care, judgment, or community. But when designed carefully and respectfully, AI can help describe, read, and explain information that would otherwise remain inaccessible.

The goal of The Dabom Project is not simply to add AI to accessibility. The goal is to make AI serve accessibility in a practical, respectful, and user-centered way.

Guiding Values

  • Accessibility before decoration.
  • Practical usefulness before technical complexity.
  • Clear spoken output before impressive but confusing responses.
  • Respect for blind users as capable, independent decision-makers.
  • Continuous improvement through real user feedback.

AI 접근성이 중요한 이유

일상의 많은 작업이 여전히 시각 정보에 크게 의존합니다 — 인쇄된 문서, 안내판, 상품 라벨, 사용자 인터페이스, 시각적 환경 모두 시각장애인에게 장벽이 될 수 있습니다.

AI가 사람의 배려, 판단력, 공동체를 대체할 수는 없습니다. 하지만 신중하고 존중하는 방식으로 설계된다면, AI는 그렇지 않았으면 접근 불가능했을 정보를 설명하고, 읽고, 해석하는 데 도움을 줄 수 있습니다.

The Dabom Project의 목표는 단순히 접근성에 AI를 추가하는 것이 아닙니다. AI가 실용적이고, 존중하며, 사용자 중심의 방식으로 접근성을 섬기도록 만드는 것입니다.

핵심 가치

  • 장식보다 접근성 먼저.
  • 기술적 복잡성보다 실용적 유용성 먼저.
  • 인상적이지만 혼란스러운 응답보다 명확한 음성 출력 먼저.
  • 시각장애인을 유능하고 독립적인 의사 결정자로 존중.
  • 실제 사용자 피드백을 통한 지속적인 개선.

Accessibility Principles

Accessibility is not a feature added to Dabom and Matheon. It is their foundation. Both applications and this website are built to be fully usable by people who rely on screen readers, keyboard navigation, speech output, and non-visual workflows.

Website Design Principles

  • Clear and predictable navigation structure
  • Meaningful button and link names
  • Proper heading hierarchy (no skipped levels)
  • Keyboard-friendly operation throughout
  • Visible focus indicators on all interactive elements
  • Skip-to-content link at the top of every page
  • Plain language wherever possible
  • No visual-only instructions

As the project grows, accessibility feedback from real users will guide every future improvement.

접근성 원칙

접근성은 다봄과 마테온에 추가된 기능이 아닙니다. 그것이 바로 두 프로그램의 토대입니다. 두 응용 프로그램과 이 웹사이트 모두 화면 낭독기, 키보드 탐색, 음성 출력, 비시각적 워크플로우에 의존하는 분들이 완전히 사용할 수 있도록 만들어졌습니다.

웹사이트 설계 원칙

  • 명확하고 예측 가능한 탐색 구조
  • 의미 있는 버튼 및 링크 이름
  • 올바른 제목 계층 구조 (단계 건너뜀 없음)
  • 전체 키보드 탐색 지원
  • 모든 상호작용 요소에 포커스 표시
  • 페이지 최상단의 본문 바로 가기 링크
  • 가능한 한 평이한 언어 사용
  • 시각 전용 안내 없음

프로젝트가 성장함에 따라, 실제 사용자의 접근성 피드백이 모든 미래 개선 사항을 안내할 것입니다.

Community and Feedback

Dabom and Matheon will improve most meaningfully when shaped by the experiences of real users. Feedback from blind and visually impaired users, educators, accessibility testers, developers, and families is always welcome.

Useful feedback may include comments about screen reader compatibility, keyboard navigation, OCR accuracy, spoken output, installation experience, documentation clarity, and real-world use cases.

The Dabom Project aims to grow as a sincere, accessibility-centered community that listens carefully and improves steadily.

커뮤니티와 피드백

다봄과 마테온은 실제 사용자의 경험으로 가장 의미 있게 발전합니다. 시각장애인 사용자, 교육자, 접근성 테스터, 개발자, 가족 분들의 피드백을 항상 환영합니다.

유용한 피드백의 예: 화면 낭독기 호환성, 키보드 탐색, OCR 정확도, 음성 출력, 설치 경험, 문서 명확성, 실제 사용 사례.

The Dabom Project는 경청하고 꾸준히 개선하는 진지한 접근성 중심 커뮤니티로 성장하고자 합니다.
